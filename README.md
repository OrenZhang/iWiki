## 部署

由于 iWiki 为前后端分离开发与部署项目，并使用到了腾讯云的部分服务，首次部署的步骤、耗时较多，可以邮件至 `incv2020@outlook.com` 获得帮助，也可点此查看[部署方法](https://wiki.incv.net/doc/291)。
